Результаты ЕГЭ по русскому языку 2012

 

Информация о проведении ЕГЭ по русскому языку в Ярославской области в 2012 году (31.05.2012г)
Статистика проведения ЕГЭ по русскому языку
31 мая 2012 года ЕГЭ по русскому языку на территории Ярославской области  сдавали 6628 человек.
Для организации ЕГЭ по русскому языку было подготовлено 56 пунктов проведения экзамена, задействовано 494 аудиторий.
Осуществляли наблюдение за проведением ЕГЭ по русскому языку 100 общественных наблюдателей.
