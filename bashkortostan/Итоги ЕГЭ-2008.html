Итоги ЕГЭ-2008

Этап государственной (итоговой) аттестации выпускников общеобразовательных учреждений

    В текущем учебном году – переходном в эпопее ЕГЭ – на территории нашей республики ЕГЭ проводился по восьми общеобразовательным предметам: русскому языку (впервые), математике, физике, химии, биологии (возобновился), географии (впервые), английскому языку (впервые), башкирскому языку (впервые). Два предмета из перечисленных были обязательными для сдачи в форме и по материалам ЕГЭ (русский язык и математика), остальные предметы сдавались по выбору самих выпускников. Особенностью этого стало то, что при выборе какого-либо из предметов, вошедших в перечень предметов ЕГЭ, выпускники должны были сдать его только в этом формате, то есть, если, например, выпускник выбрал в качестве выпускного экзамена физику, то он сдавал её только в формате ЕГЭ, экзаменов в традиционной форме по названным предметам не проводилось, кроме как для выпускников, обучавшихся в щадящем режиме. Ниже представлен краткий анализ по каждому из предметов ЕГЭ в хронологическом порядке их проведения. Более подробный анализ будет представлен в августе-сентябре 2008 года в полиграфическом варианте.

    АНГЛИЙСКИЙ ЯЗЫК (16 мая 2008 г.)
    Открывал череду единых государственных экзаменов в 2008 году экзамен по английскому языку. В нём приняли участие 299 выпускников общеобразовательных учреждений, выпускников учреждений НПО и СПО и выпускников прошлых лет Республики Башкортостан. Сопоставление республиканских результатов с общероссийскими свидетельствует, что знания, показанные нашими выпускниками, чуть ниже общероссийских, но вполне с ними сопоставимы: 62,8 – средний балл по Республике Башкортостан, 64,8 – средний балл по РФ.
    Единый государственный экзамен по английскому языку в нашей республике проводился впервые, поэтому мы не можем сравнивать результаты ЕГЭ-2008 с результатами ЕГЭ-2007 по данному предмету. В приведенной ниже таблице предлагаются статистические данные о результатах ЕГЭ по республике и стране.

    ХИМИЯ (19 мая 2008 г.)
    В едином государственном экзамене по химии принял участие 1 441 выпускник из общеобразовательных учреждений, учреждений НПО и СПО Республики Башкортостан. Ежегодно средние результаты республиканских выпускников несколько выше средних результатов выпускников школ России, не стал исключением и ЕГЭ-2008 по химии: 62,5 – средний балл по Республике Башкортостан, 55,5 – средний балл по РФ.     БИОЛОГИЯ (26 мая 2008 г.)
    В едином государственном экзамене по биологии приняли участие 1 639 выпускников из общеобразовательных учреждений, учреждений НПО и СПО Республики Башкортостан. Биология в формате ЕГЭ сдается нашими выпускниками не впервые, однако сравнивать результаты этого года и предыдущего не получается: прошло довольно много времени с тех пор, как республиканские выпускники и абитуриенты экзаменовались в данном формате по биологии, с тех пор несколько видоизменились как сами контрольно-измерительные материалы, так и система шкалирования. В приведенной ниже таблице предлагаются статистические данные о результатах ЕГЭ-2008 по биологии по республике и стране в целом.

В этом году наивысшие баллы в ЕГЭ по биологии не набрал ни один школьник из нашей республики.

    РУССКИЙ ЯЗЫК (29 мая 2008 г.)
    Единый государственный экзамен по русскому языку – один из двух обязательных (наряду с математикой) для сдачи всеми выпускниками 2008 года. Он проводился в нашей республике впервые, для многих стал неожиданностью (о его обязательности узнали лишь во второй половине марта). Возможно, именно этим объясняются несколько более низкие результаты ЕГЭ-2008 по русскому языку в сравнении со средними результатами в России.     БАШКИРСКИЙ ЯЗЫК (16 июня 2008 г.)
    Единый государственный экзамен по башкирскому языку в таком режиме проводился в республике впервые (в прошлом году ЕГЭ по башкирскому языку проводился лишь в режиме репетиции), поэтому сравнивать его результаты с прошлым годом нет возможности. Тем более, нет возможности сравнивать с результатами по России, так как это региональный экзамен. Можно было бы сравнить с результатами ЕГЭ по русскому языку: эти два предмета выступают в качестве альтернативных для поступления на некоторые специальности в некоторые вузы, однако русский язык сдавали все выпускники в обязательном порядке, а башкирский - по выбору. Исходя из сказанного выше, предлагаем статистические данные результатов ЕГЭ-2008 по башкирскому языку без какого-либо сравнения.

    В едином государственном экзамене по башкирскому языку принял участие 751 выпускник из районов и городов республики. Так как данный предмет выступает в качестве альтернативного русскому языку для поступления на некторые специальности в некоторые вузы, разработчики контрольно-измерительных материалов старались приблизить сам набор заданий и их уровень сложности к названному федеральному предмету. Шкалирование по проценту выполненных заданий также идентично русскому языку. Несколько более высокие результаты по башкирскому языку в сравнении с русским языком обусловлены выборностью данного предмета: на экзамен пришли только те выпускники, которые не сомневаются в своих хороших знаниях предмета.

    РЕЗЕРВНЫЕ ДНИ
    В резервные дни экзамены сдавали по всем предметам, кроме башкирского языка (по данному предмету был указан отдельный день резерва, но сдавать ЕГЭ в этот день не пожелал никто). Так как количество экзаменуемых в дни резерва (10 и 11 июня 2008 года) относительно основных дней мало, предлагаем лишь сводную ведомость результатов по этим дням.
